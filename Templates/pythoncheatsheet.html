Python Cheat Sheet

Basic Syntax

Print:

print("Hello, World!")

Variables:

x = 10
name = "Alice"

Lists:

my_list = [1, 2, 3, 4]

Dictionaries:

my_dict = {"key": "value"}

Control Structures

If Statement:

if x > 10:
  print("x is greater than 10")
elif x == 10:
  print("x is 10")
else:
  print("x is less than 10")

For Loop:

for i in range(5):
  print(i)

While Loop:

while x > 0:
  print(x)
  x -= 1

Functions

Function Definition:

def greet(name):
  return f"Hello, {name}"

Lambda Functions:

square = lambda x: x ** 2
print(square(5)) # Output: 25

Common Libraries

Math Library:

import math
print(math.sqrt(16)) # Output: 4.0

Datetime Library:

from datetime import datetime
now = datetime.now()
print(now)
